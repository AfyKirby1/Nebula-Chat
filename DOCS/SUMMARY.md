## The "Why" (Value Proposition)

Why use Nebula Chat over the standard Google AI Studio or other interfaces?

1.  **Transparency**: See the "Thinking" process of the model (Gemini 3 Pro).
2.  **Control**: You own the API key, the history (local), and the configuration.
3.  **Visuals**: The Neural Canvas provides a unique, immersive aesthetic that reduces fatigue.
4.  **Workflow**: The Artifact Viewer allows for side-by-side coding and chatting, mimicking an IDE experience.
5.  **Multimodal**: Talk to your AI and show it images, just like a human collaborator.
6.  **Education**: It's a teaching tool for how to build streaming AI apps with Next.js.
